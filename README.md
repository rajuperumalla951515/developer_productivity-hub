# Developer Productivity Hub

A React + Vite MVP for tracking developer productivity, team analytics, and insights with a polished dashboard UI.

## Tech Stack

- React
- Vite
- Tailwind CSS

## Features

- Developer selector with product ownership and email details
- Team analytics and insights dashboards
- Responsive, premium-style UI with smooth page transitions

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
